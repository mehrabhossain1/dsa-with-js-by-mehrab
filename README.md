# DSA with JavaScript by Mehrab

A structured, public learning repository where I document my journey to improve problem-solving skills using JavaScript.

This is not just a collection of solutions — it’s a **learning system** focused on understanding patterns, optimizing approaches, and building consistency.

---

## 🚀 Objectives

- Strengthen core Data Structures & Algorithms fundamentals
- Become confident solving medium-level problems on LeetCode
- Develop strong problem-solving intuition
- Learn in public and help others follow along

---

## 🧠 Learning Approach

Each concept or problem in this repository follows a consistent structure:

- **Implementation** — Clean and readable JavaScript code
- **Explanation** — Step-by-step breakdown of the approach
- **Complexity Analysis** — Time and space complexity
- **Mistakes & Learnings** — What I got wrong and how I improved

> Philosophy: _Learn → Apply → Reflect → Repeat_
